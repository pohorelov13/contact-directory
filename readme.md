## Опис
Цей проект представляє собою консольний довідник для ведення списку компаній та контактної інформації. Довідник дозволяє додавати компанії разом із контактними особами і номерами телефонів, а також видаляти компанії та їх контактні дані. Проект реалізований на Java версії 17 та використовує базу даних MySQL. Також використано Spring Boot, Data JPA та Maven. Ліцензовано Apache License 2.0.

## Функціональність
- Виведення списку компаній та контактної інформації.
- Додавання нових компаній з контактними особами та номерами телефонів.
- Видалення компаній та їх контактної інформації.
